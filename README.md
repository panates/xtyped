Extended type schema library for javascript.

  - Extensible Types
  - Generic Types
  - Safe data type conversion between Javascript and JSON
  - Data validation
  - Easy type definition with strings
  - 10 built-in types  
  - Tested with more than 180 unit tests

## Installation

`$ npm install xtyped --save`
  
## Node Compatibility

  - node `>= 0.10`;
